   27  5-10-5 TEST
This test is also called the pro agility test or the 20-yard shuttle.
Equipment
An American football field or other field marked with three parallel lines 5 yards (4.6 m) apart
(figure 14.23)
Stopwatch or timing gates, if available, with corresponding software on a laptop, tablet, or
smartphone application
Personnel
One timer/recorder, one line judge
Procedure
1. If using one timing gate for total end time, set this up at the start and finish line. If multiple
gates are available, set up at the start and finish line as well as at each turn line to measure
each split.
2. The athlete straddles the center line using a three-point stance.
3. On an auditory signal, the athlete sprints 5 yards (4.6 m) to the line on the left, then changes
direction and sprints 10 yards (9 m) to the line on the right, then again changes direction and
sprints 5 yards (4.6 m) to the center line. Hand (or foot) contact must be made with all
indicated lines. (Note: It is important that this is kept consistent for both trials.)
4. The best time of two trials is recorded to the nearest 0.01 second.
Note: Normative data for the 5-10-5 test (previously pro agility) are presented in tables 14.29 and
14.30 near the end of the chapter.
