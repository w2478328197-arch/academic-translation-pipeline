   45   CLEAN PULL FROM KNEE-BLOCK
(AND SNATCH GRIP VARIATION)
The clean pull from the knee-block (and snatch grip variation) is an
exercise that is performed from a position that corresponds to the
end of the first pull during the performance of the power clean (or
power snatch). The start position corresponds to the end of the
downward movement of the RDL exercise.
Starting Position
Position the blocks so that the loaded barbell is at a height that
corresponds to the athlete’s patellar tendon.
Stand with the feet placed between hip- to shoulder-width apart
with the toes pointed slightly outward.
Grasp the bar with a clean grip (i.e., slightly wider than
shoulder-width apart) while using lifting straps with the elbows
fully extended and pointing outward (i.e., arms internally
rotated and the shoulder retracted). (If performing this exercise
