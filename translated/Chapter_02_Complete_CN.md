### 骨骼肌系统
为了产生运动或对外部物体产生力，每块骨骼肌的两端必须通过结缔组织附着在骨骼上。传统上，解剖学家将肌肉的**起点（origin）**定义为其近端（靠近身体中心）附着点，将**止点（insertion）**定义为其远端（远离身体中心）附着点。有时，起点被定义为肌肉附着点中相对固定的结构，而止点定义为活动性较大的结构。这种定义有时会导致起点和止点的混淆。例如，在直腿仰卧起坐中，髂肌的起点是股骨，因为其相对固定。骨盆由于活动性较大，成了止点。然而，在抬腿练习中，骨盆相对固定，因此成为起点，而活动性较大的股骨则成为止点。因此，传统定义提供了最高的一致性。

肌肉以各种方式附着在骨骼上。在**肉质附着（fleshy attachments）**（多见于肌肉的近端）中，肌纤维直接固定在骨骼上，通常分布在较宽的区域，从而使力得以分散而非集中。**纤维附着（fibrous attachments）**（如**腱/肌腱**）与肌肉鞘和骨骼周围的结缔组织融合并连续。它们还有延伸到骨骼内部的纤维，形成非常牢固的结合。

几乎所有的身体运动都涉及多块肌肉的共同作用。与产生运动最直接相关的肌肉被称为**原动肌（prime mover）**或**主动肌（agonist）**。能够减慢或停止运动的肌肉被称为**对抗肌（antagonist）**。对抗肌协助关节稳定，并在快速运动结束时起到制动肢体的作用，从而保护韧带和软骨关节结构免受潜在的破坏力。例如，在投掷过程中，三头肌作为主动肌使肘部伸展以加速球。当肘部接近完全伸展时，二头肌作为对抗肌减慢肘部伸展并使其停止，从而保护肘部结构免受内部冲击。

<!-- Page 107 -->
当一块肌肉间接协助运动时，被称为**协同肌（synergist）**。例如，在手臂运动期间，稳定肩胛骨的肌肉起到了协同肌的作用。如果没有这些协同肌，移动上臂的肌肉（其中许多起源于肩胛骨）将无法有效地产生这种运动。当主动肌是跨越两个关节的肌肉时，也需要协同肌来控制身体运动。例如，股直肌跨越髋关节和膝关节，收缩时起到屈髋和伸膝的作用。从深蹲中站起涉及髋关节和膝关节的伸展。如果股直肌要在人不使躯干前倾的情况下发挥伸膝作用，那么像臀大肌这样的伸髋肌必须发挥协同作用，以抵消股直肌张力可能导致的屈髋作用。

### 肌肉骨骼系统的杠杆
虽然体内有许多肌肉不通过杠杆发挥作用（如面部、舌头、心脏、动脉和括约肌的肌肉），但直接涉及体育和健身的身体运动主要通过骨骼系统的骨杠杆发挥作用。为了理解身体如何影响这些运动，需要具备杠杆的基本知识。以下是几个基本定义。

**第一类杠杆（first-class lever）**：肌力和阻力作用在支点两侧的杠杆（见图 2.1）。

<!-- Page 108 -->
![图 2.1](images_chapter2/page_108_img_0.jpeg)
图 2.1 第一类杠杆（前臂）。克服阻力伸肘（例如，三头肌伸展练习）。O = 支点；FM = 肌力；FR = 阻力；MM = 肌力的力臂；MR = 阻力的力臂。机械优势 = MM/MR = 5 cm / 40 cm = 0.125，小于 1.0，这在通常意义上是劣势。

**支点（fulcrum）**：杠杆的转动中心。

**杠杆（lever）**：一个刚性或半刚性的物体，当受到一个作用线不通过其转动中心的力时，会对任何阻碍其转动趋势的物体产生力（见图 2.2）。

<!-- Page 109 -->
![图 2.2](images_chapter2/page_109_img_0.jpeg)
图 2.2 杠杆。杠杆可以将切向于旋转弧的力从物体长度上的一个接触点传递到另一个接触点。FA = 施加于杠杆的力；MAF = 施力力臂；FR = 阻碍杠杆转动的力；MRF = 阻力力臂。杠杆对物体施加的力大小与 FR 相等，但方向相反。

**机械优势（mechanical advantage）**：施力力臂与阻力力臂的比率。为了使施加转矩和阻力转矩之间达到平衡状态，肌力与其力臂的乘积必须等于阻力与其力臂的乘积。因此，机械优势（即比率大于 1.0）允许施加的（肌肉）力小于阻力即可产生相等的转矩。相反，如果计算出的机械优势比率小于 1.0，则意味着必须施加比阻力更大的（肌肉）力，这显然对肌肉不利。

**力臂（moment arm）**（也称为力量臂、杠杆臂或转矩臂）：从力的作用线到支点的垂直距离。力的作用线是一条通过力的施加点并沿力作用方向延伸的无限长直线。

**肌力（muscle force）**：由生化活动或非收缩组织的拉伸产生的力，倾向于将肌肉的两端拉向彼此。

<!-- Page 110 -->
**阻力（resistive force）**：由身体外部来源（如重力、惯性、摩擦力）产生的、与肌力方向相反的力。

**第二类杠杆（second-class lever）**：肌力和阻力作用在支点同侧的杠杆，且肌力的力臂长于阻力的力臂。例如当小腿肌肉工作使身体由脚掌支撑抬起时（见图 2.3）。由于其机械优势（即相对较长的力臂），所需的肌力小于阻力（体重）。

**第三类杠杆（third-class lever）**：肌力和阻力作用在支点同侧的杠杆，且肌力的力臂短于阻力的力臂（见图 2.4）。因此，机械优势小于 1.0，所以肌力必须大于阻力，才能产生与阻力相等的转矩。

**转矩（torque）**（也称为力矩）：力使物体绕特定支点转动的程度。定量定义为力的大小乘以其力臂的长度。

![图 2.3](images_chapter2/page_111_img_0.jpeg)
图 2.3 第二类杠杆（足部）。克服阻力进行跖屈（例如，站姿提踵练习）。FM = 肌力；FR = 阻力；MM = 肌力的力臂；MR = 阻力的力臂。当身体抬起时，足尖（脚掌）作为足部转动的点，即为支点（O）。因为 MM 大于 MR，所以 FM 小于 FR。

图 2.1 显示了一个第一类杠杆，因为肌力和阻力作用在支点的两侧。在等长用力或恒速关节转动期间，FM × MM = FR × MR。由于 MM 远小于 MR，FM 必须远大于 FR；这说明了这种排列的劣势性质（即需要很大的肌力来对抗相对较小的外部阻力）。

![图 2.4](images_chapter2/page_112_img_0.jpeg)
图 2.4 第三类杠杆（前臂）。克服阻力屈肘（例如，二头肌弯举练习）。FM = 肌力；FR = 阻力；MM = 肌力的力臂；MR = 阻力的力臂。由于 MM 远小于 MR，FM 必须远大于 FR。

大多数使肢体绕关节转动的人体肌肉都在小于 1.0 的机械优势下工作（即机械劣势）。这就是为什么内部肌力远大于身体对外部物体施加的力。例如在图 2.2 中，由于阻力力臂比肌力力臂长 8 倍，肌力必须是阻力的 8 倍。肌肉和肌腱承受的极高内部压力是导致这些组织损伤的主要原因。在实际运动中，将杠杆分类为第一、第二或第三类通常取决于对支点位置的某种程度的武断决定。因此，理解机械优势原理比能够对杠杆进行分类重要得多。
